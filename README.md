# Mouse Tracker WinForms

Приложение на C# Windows Forms для отслеживания позиции мыши и обработки нажатий кнопок.

## Функциональность

### 🖱️ Отслеживание мыши
- При движении мыши по форме:
  - Координаты отображаются в textBox1
  - Координаты отображаются в заголовке окна

### 🖱️ Обработка нажатий на кнопку
- При нажатии на btnPosition:
  - Левая кнопка → сообщение "Левая кнопка мыши!"
  - Правая кнопка → сообщение "Правая кнопка мыши!"

### 🔚 Завершение работы
- Кнопка btnClose закрывает приложение

## Интерфейс

- textBox1 — отображение текущих координат мыши
- btnPosition — кнопка для проверки нажатий
- btnClose — кнопка выхода

## Пример работы

1. При движении мыши: textBox1 показывает "(150, 75)", заголовок окна обновляется
2. Нажатие левой кнопки на btnPosition → сообщение "Левая кнопка мыши!"
3. Нажатие правой кнопки на btnPosition → сообщение "Правая кнопка мыши!"

## Куда можно встроить

- В демонстрационные приложения
- В обучающие проекты по работе с мышью
- В примеры обработки событий MouseMove и MouseDown
- В отладочные инструменты

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

Событие MouseMove срабатывает при движении мыши по всей форме. Для btnPosition обрабатываются только нажатия, движение не отслеживается.
